# Installation de mviewerstudio

mviewer studio peut fonctionner avec 3 backends différents
 * apache2 + php7 activé
 * python flask
 * Docker

En fonction du backend retenu, l'installation diffère.

## Cas 1 apache2 + php7

Prerequis
----------

    apache2 + php7


clone du projet
---------------

* `cd /home/studio`
* `git clone https://github.com/geobretagne/mviewerstudio`
